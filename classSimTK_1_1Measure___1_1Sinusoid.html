Detailed Description

template<class T>
class SimTK::Measure_< T >::Sinusoid

This measure produces a sinusoidal function of time: 

     m(t) = a*sin(w*t+p)

where a=amplitude in arbitrary units, w=frequency in rad/unit time, p=phase in radians.
